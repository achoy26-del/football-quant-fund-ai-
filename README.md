# Football Quant Fund AI

Institutional-style football betting research knowledge base for Gemini, Claude, and ChatGPT.

This project is designed to help an AI assistant price football markets, identify positive expected value opportunities, control risk, and produce disciplined match reports.

Important: this project does **not** claim to reproduce Tony Bloom, StarLizard, Pinnacle, SBO, or any proprietary betting model. It uses public, explainable methods and strict risk controls.

## Structure

- `docs/` — 30 Markdown manuals
- `knowledge/` — reference notes for models and concepts
- `templates/` — reusable report and journal templates
- `prompts/` — Gemini, Claude, and ChatGPT setup prompts
- `.github/workflows/` — optional PDF build workflow

## Core principle

No edge, no bet.
