# Poisson Goal Model

## Purpose
Estimate the probability of goal counts from expected goal rates.

## Summary
For each team, estimate expected goals, then calculate probabilities for 0, 1, 2, 3+ goals.

## Use
- 1X2 probabilities
- Over/Under probabilities
- Correct score distribution

## Limitation
Football goals are not perfectly independent. Use adjustments for low scores, tactics, and match state.
