# 14 Value Betting

## Purpose
Value betting means buying a price that is higher than fair value.

## Summary Formula
Expected Value = (True Probability × Decimal Odds) - 1

## Required Conditions
- True probability is independently estimated.
- Market odds are available at or above target entry.
- Risk is acceptable.

## Minimum Thresholds
- Edge above uncertainty buffer
- Positive expected ROI
- Liquidity sufficient for execution

## Rule
Value is price-dependent. The same selection can be value at 2.10 and no value at 1.85.
