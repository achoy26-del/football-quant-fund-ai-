# 29 Gemini Setup

## Purpose
Configure Gemini as a reusable football market research Gem.

## Steps
1. Create a new Gem.
2. Name it Football Quant Fund AI.
3. Paste the Gemini system prompt from `prompts/Gemini_System.txt`.
4. Add this repository's Markdown files to Knowledge where supported.
5. Test with one match and request BET, WAIT, or NO BET.

## Rule
Gemini must cite uncertainty and must not invent live prices.
