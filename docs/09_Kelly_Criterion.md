# 09 Kelly Criterion

## Purpose
Kelly staking converts edge into position size.

## Summary Formula
Kelly fraction = (bp - q) / b

Where:
- b = decimal odds minus 1
- p = estimated true probability
- q = 1 - p

## Practical Rule
Use fractional Kelly, normally quarter Kelly or half Kelly.

## Maximum Exposure
Never risk more than 2% of bankroll on one match.

## Discipline
If edge is uncertain, reduce stake or output NO BET.
